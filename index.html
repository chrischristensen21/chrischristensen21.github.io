Weather and Latitude Analysis
The purpose of this website is to display the results of a weather analysis.

For the analysis, data from over 500 cities was extracted from OpenWeatherMap API. This data included several different weather-related metrics which included: max temperature, percent cloudiness, wind speed and humidity. The goal of the analysis was to see how these weather metrics are affected as you approach the equator. Once the data was collected, various analyses were performed, and visualizations created using matplotlib.

The data used to create the visualizations is available to view in the “Data” tab. You can also view a list of each visualization created under the “Plots” drop-down menu. This will take you to an in-depth analysis of each visualization. In addition to the drop-down menu, you can also go straight to your desired analysis by clicking the figure in the visualization area that corresponds to the analysis you would like to view. If you would like to see a side-by-side comparison of each visualization you can view this in the “Comparisons” tab. Finally, may your weather experience be a breeze!
